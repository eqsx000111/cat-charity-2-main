# Cat Charity API

Благотворительный API‑сервис для управления целевыми проектами и пожертвованиями.
Проект реализован на **FastAPI + SQLAlchemy (async) + Alembic** и поддерживает автоматическое распределение пожертвований по проектам.

---

## Описание проекта

Cat Charity — это REST API, который позволяет:
* производить регистрацию и авторизацию пользователей
* создавать благотворительные проекты с целевой суммой;
* принимать пожертвования;
* **автоматически инвестировать пожертвования в проекты**;
* закрывать проекты при достижении целевой суммы;

Инвестирование происходит **автоматически**:

* при создании проекта — если в системе есть нераспределённые пожертвования;
* при создании пожертвования — если есть открытые проекты.

Проекты инвестируются **в порядке создания**.

## Запуск проекта

### 1. Клонировать репозиторий

```bash
git clone https://github.com/eqsx000111/cat-charity-2-main
cd cat-charity-2-main
```

### 2. Создать и активировать виртуальное окружение

```bash
python -m venv venv
source venv/bin/activate  # macOS / Linux
venv\Scripts\activate     # Windows
```

### 3. Установить зависимости

```bash
pip install -r requirements.txt
```

### 4. Настроить переменные окружения

Создайте файл `.env`:

```env
DATABASE_URL=sqlite+aiosqlite:///./cat_charity.db
```
